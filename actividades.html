Pregunta Verdadero-Falso

Lee las siguientes afirmaciones y selecciona si son Verdaderas o Falsas. Al finalizar, recibirás una retroalimentación automática.

Pregunta 1

Subir una foto a redes sociales implica seguir un algoritmo.

Retroalimentación

Verdadero

¡TEN EN CUENTA!

Una app sigue pasos definidos para subir y mostrar la foto.

Pregunta 2

Un algoritmo solo se usa en programación y no en la vida diaria.

Retroalimentación

Falso

¡RECUERDA!

Usamos algoritmos también en tareas cotidianas.

Pregunta 3

Hornear un pastel siguiendo una receta es un ejemplo de algoritmo.

Retroalimentación

Verdadero

¡TEN PRESENTE!

Una receta es una secuencia de pasos, como cualquier algoritmo.

Pregunta 4

Para crear un algoritmo siempre necesitas una computadora.

Retroalimentación

Falso

¡MUY BIEN!

Puedes crear algoritmos con papel y lápiz, sin tecnología.

Lista desordenada

Organiza los pasos correctamente para formar un algoritmo.

  • Input: Un zapato con cordones desatados. Tus dos manos.
  • Coloca el zapato en tu pie.
  • Toma un cordón con cada mano.
  • Cruza los cordones y tira para ajustarlos.
  • Haz un lazo con el cordón derecho.
  • Envuelve el lazo con el cordón izquierdo.
  • Pasa el cordón izquierdo por el agujero que se forma.
  • Tira de ambos lazos para ajustar el nudo.
  • Asegúrate de que el lazo esté firme.
  • Output: Zapato firmemente atado y listo para caminar.

Comprobar

¡Correcto! Has comprendido que un algoritmo necesita un input claro, una secuencia lógica de pasos y un output específico. Tu diseño muestra que sabes organizar tareas como una serie de instrucciones precisas.
