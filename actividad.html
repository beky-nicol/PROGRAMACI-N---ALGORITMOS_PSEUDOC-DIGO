Crea tu primer Pseudocódigo

Es hora de poner en práctica lo que hemos aprendido. El pseudocódigo es nuestra forma de planificar y describir los pasos para resolver un problema antes de escribir código real.

El Reto: Sumar Dos Números

Tu tarea es escribir el pseudocódigo para un programa sencillo que haga lo siguiente:

  • Pedir al usuario dos números
  • Sumar los números
  • Mostrar el resultado


Indicaciones: 

Usa las convenciones que hemos estado estudiando para definir variables, pedir entrada, realizar operaciones, mostrar salida

No te preocupes por la sintaxis perfecta de un lenguaje de programación real. Concéntrate en la lógica y la secuencia de los pasos.

Escribe

HOLA

Proceso SumarDosNumeros
// Definir las variables que vamos a usar
Definir numero1, numero2, resultado Como Entero

// Pedir al usuario el primer número
Escribir "Por favor, ingresa el primer número:"
Leer numero1

// Pedir al usuario el segundo número
Escribir "Por favor, ingresa el segundo número:"
Leer numero2

// Realizar la suma
resultado = numero1 + numero2

// Mostrar el resultado
Escribir "La suma de los dos números es: ", resultado
FinProceso
